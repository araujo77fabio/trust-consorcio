# Consórcio Novacapu - Site Institucional

Site institucional profissional demonstrando as vantagens do consórcio sobre o financiamento tradicional através de dados, gráficos e análises matemáticas. Design inspirado no Synoro Invest.

## 🚀 Características

- **Design Institucional Moderno** inspirado em sites financeiros de alto padrão
- **13 Seções Completas** com conteúdo rico e dados reais
- **Gráficos Interativos** usando Recharts para visualização de dados
- **Animações Profissionais** com Framer Motion
- **Totalmente Responsivo** com TailwindCSS
- **Navegação Intuitiva** com menu fixo e scroll suave

## 📊 Estrutura do Site

1. **Hero** - Apresentação impactante com estatísticas principais
2. **A Nova Era do Patrimônio** - Introdução à engenharia financeira
3. **O Custo do "Quero Agora"** - Comparativo financiamento vs consórcio (R$ 1.350.000 vs R$ 600.000)
4. **Comparativo de Fluxo de Caixa** - Análise de parcelas mensais e investimento da diferença
5. **A Matemática da Taxa de Administração** - CET: 1,6% vs 11,5% ao ano
6. **Alavancagem com Aluguel** - Imóvel que se paga sozinho (aporte de apenas R$ 300/mês)
7. **Estratégia de Lance Embutido** - Rodobens: 75% recursos + 25% crédito
8. **ROI na Venda da Carta** - Rentabilidade de 140% sobre capital investido
9. **O Poder da Compra à Vista** - Descontos de 5-15% que compensam a taxa
10. **Plano Pontual e Previsibilidade** - Prazos garantidos e contemplação programada
11. **Proteção Patrimonial** - INCC/IPCA: valorização de R$ 500k para R$ 660k
12. **Solidez em Números** - Novacapu: +35 anos, +500k clientes, R$ 10Bi+
13. **Call to Action** - Contato e simulação com QR Code

## 🛠️ Tecnologias Utilizadas

- React 18
- Vite
- TailwindCSS
- Recharts (gráficos)
- Framer Motion (animações)
- Lucide React (ícones)

## 📦 Instalação

```bash
npm install
```

## 🚀 Executar em Desenvolvimento

```bash
npm run dev
```

O site estará disponível em `http://localhost:3000`

## 🏗️ Build para Produção

```bash
npm run build
```
